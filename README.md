🍃 Sri Lanka Tea Industry Analysis Dashboard

> An interactive Power BI dashboard that analyzes Sri Lanka's tea industry using production, export, regional, and environmental data to generate actionable business insights.

📖 Project Overview

The Sri Lanka Tea Industry Analysis Dashboard was developed using Microsoft Power BI to provide a comprehensive analysis of tea production across different regions of Sri Lanka.

The dashboard enables users to explore production trends, regional performance, export metrics, and environmental factors such as rainfall through interactive visualizations and KPIs.

This project demonstrates data modeling, DAX calculations, dashboard design, and storytelling techniques commonly used by Business Intelligence and Data Analysts.

🎯 Objectives

- Analyze tea production across Sri Lankan regions.
- Compare regional performance and contribution.
- Identify production trends over time.
- Explore the relationship between rainfall and tea production.
- Provide an interactive dashboard for business decision-making.

 🛠 Tools & Technologies

- Microsoft Power BI
- Microsoft Excel
- Power Query
- DAX (Data Analysis Expressions)

📊 Dashboard Features

 📌 Executive Dashboard

- Total Production KPI
- Total Export Value KPI
- Production Trend Analysis
- Regional Production Overview
- Interactive Filters

🌍 Regional Performance Analysis

- Geographic Distribution Map
- Tea Production by Region
- Production Achievement Gauge
- Tea Production Composition by Tea Type
- Regional Performance Table

📈 Key Performance Indicators (KPIs)

- Total Tea Production (MT)
- Total Export Value (MUSD)
- Average Production
- Regional Contribution
- Production Growth
- Production Achievement
